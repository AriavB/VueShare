## VueShare
Pinterest inspired full stack app with Vue, GraphQL, Apollo , Vuex, and Vuetify. Share your photos with the community. Complete CRUD and Search functionality with MongoDB. NodeJS Server.

## Client
Vue

```
npm install
```
